# VDI Project - Data Visualization Dashboard

A data visualization project displaying global statistics and trends.

## Datasets

This project uses the following datasets:

- **GDP per Capita** - World Bank data on GDP per capita across countries
- **Life Expectancy** - Life expectancy statistics by country
- **School Enrollment** - Tertiary education enrollment rates
- **Unemployment Rate** - Youth unemployment data

## Files

- `index.html` - Main HTML file
- `script.js` - JavaScript for data visualization
- `styles.css` - Styling

## How to Run

1. Open `index.html` in a web browser
2. The visualizations will load automatically

## Data Sources

All datasets are sourced from reputable international organizations and are included in their respective folders.
